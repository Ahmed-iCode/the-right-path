# تطبيق إسلامي - تطبيق العبادة اليومية

تطبيق React لتحفيز العبادة اليومية مع نظام الحسنات والبادجات.

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm start
```

## البنية

- **الشريط العلوي**: عرض اللقب، المستوى، وشريط الحسنات (XP)
- **منطقة المحتوى**: تتغير حسب التبويب المختار
- **شريط التنقل السفلي**: 5 تبويبات رئيسية (الرئيسية، الرحلة، المكتبة، الأدوات، حسابي)
